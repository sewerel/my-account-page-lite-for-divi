== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` or install via the WordPress Plugin Installer.
2. Activate the plugin through the ‚ÄòPlugins‚Äô screen.
3. Go to **Divi MyAccount Page** in the WordPress Admin to configure the order, icons, names, and default endpoint.

== Frequently Asked Questions ==

= Does this plugin require Divi? =
Yes, Divi is required to fully customize endpoint pages using the Divi Builder.

= Can I create new endpoints with this version? =
No. Endpoint creation is a Pro-only feature. This Lite version lets you rename, reorder, and style existing endpoints.

= Can I hide menu items for different user roles? =
That feature is included in the Pro version.

== Source Code and Build Instructions ==

The human-readable (non-minified) JavaScript and CSS source for this plugin
is publicly available here:

https://github.com/yourusername/sewerel

To build the production assets used in the distributed plugin:

1. Clone the repository:
   git clone https://github.com/sewerel/my-account-page-lite-for-divi.git

2. Install dependencies:
   npm install

3. Build production assets:
   npm run build

This process generates the files located in /assets/js/*.min.js and /assets/css/*.min.css
that are included in the WordPress.org plugin release.
